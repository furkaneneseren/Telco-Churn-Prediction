# Telco-Churn-Prediction
In this project, I developed a model that will make prediction from telco customer data by using stacking ensemble. The dataset in this project is a public dataset from kaggle.

## Tech and Libraries

- Pandas
- Numpy
- Sklearn
- Tensorflow
- MLPClassifier
- RandomForestClassifier
- XGBClassifier
- Logistic Regression

## Dataset

 - [Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
